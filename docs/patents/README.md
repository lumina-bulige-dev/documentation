# patents フォルダ説明書

このフォルダには、**日本国特許庁（JPO）の明細書構成を意識した特許文章のドラフト**を格納しています。

## 収録内容

- `patent-01-ai-supply-chain-risk-graph.md`  
  サプライチェーン全体の脆弱性・脅威伝播をリアルタイムに評価し、被害最小化行動まで自動提案する高価値特許ドラフト。
- `patent-02-federated-private-model-adaptation.md`  
  企業横断データを秘匿しつつ高精度モデルを継続学習できる高価値特許ドラフト。

## 使い方

1. 事業領域に合わせて「発明の名称」「請求項」「実施形態」の固有名詞を差し替える。
2. 図面番号（図1〜）を社内図面と整合させる。
3. 先行技術調査の結果に応じて請求項の従属関係を調整する。

## 注意

- 本文書は出願前の叩き台です。実際の出願時には弁理士・知財部門レビューを前提としてください。
